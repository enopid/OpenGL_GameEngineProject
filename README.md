    모드 1 - 고해상도 단일 오브젝트 PBR 프로퍼티 조정 example

    마우스 - 시점조작
    Q/E - 현재 시점에서 위아래 이동
    A/D - 현재 시점에서 좌우 이동
    W/S - 현재 시점에서 앞뒤 이동
